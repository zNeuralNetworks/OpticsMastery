    Shell script for easy management
    
    Commands:
      ./docker-control.sh start     # Start container
      ./docker-control.sh stop      # Stop container
      ./docker-control.sh restart   # Restart container
      ./docker-control.sh logs      # View live logs (Ctrl+C to exit)
      ./docker-control.sh stats     # Resource usage
      ./docker-control.sh status    # Check status
      ./docker-control.sh update    # Pull latest image
      ./docker-control.sh rebuild   # Rebuild from Dockerfile
    
    Features:
      ✓ Colored output
      ✓ Error handling
      ✓ Container existence checks
      ✓ Health status verification
      ✓ Environment variable support (PORT=8080)
      ✓ Helpful messages & examples
